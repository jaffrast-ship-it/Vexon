# VEXON Streetwear E-commerce

A premium dark streetwear storefront inspired by Cyrclo Webflow template.

## Usage

- Open `index.html` in a browser
- Product detail page: `product.html?id=hoodie-black`
- Contact page: `contact.html`

## Features

- Hero video background
- Featured products with hover video
- Filtering, search, wishlist, cart
- Mobile responsive design
- Cash on Delivery checkout simulation
- About and customer-of-month sections
